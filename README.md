# Cloth Rental System

A simple web application for renting clothes online.

## What It Does

Customers can browse and rent clothes. Admins can manage products and orders.

## Built With

PHP, MySQL, HTML, CSS, JavaScript

## Installation

1. Clone this repository
2. Copy `config/db.example.php` to `config/db.php` and add your database details
3. Import `database/setup.sql` into MySQL
4. Place in your web server directory (e.g., `C:\xampp\htdocs\`)
5. Access at `http://localhost/Cloth_rental_system/`

Default admin login: `admin@admin.com` / `admin123`

## Usage

**Customers:** Register → Browse → Add to cart → Checkout

**Admins:** Login → Manage products → Process orders
